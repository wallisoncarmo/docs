---

Este guia ajuda você a dar os primeiros passos no **Main Agents Store**.

---

## Pela Interface (UI)

1. Acesse o painel do Main Agents Store.
2. Escolha um **Agente** (ex.: Contestação de Infração).
3. Faça o **upload dos documentos** exigidos.
4. Aguarde o processamento.
5. Baixe o **parecer em PDF** diretamente na interface.

> Dica: insira aqui **prints de tela** mostrando o fluxo para os usuários.

---

## Pela API

1. **Envie a tarefa** usando o endpoint de upload do agente/tarefa:

```bash
POST /main-agent/v1/agent-task/upload/{agent}/{task}
```

2. **Consulte o status** com o `execution_id` retornado:

```bash
GET /main-agent/v1/agent-message/process/{execution_id}
```

3. **Baixe o resultado** quando o status for `Completo`:

```bash
GET /main-agent/v1/agent-message/download/{execution_id}
```

---

## Exemplo completo (Contestação de Infração)

```bash
curl -X POST "https://agents-store-api.alfred.vert.com.br/main-agent/v1/agent-task/upload/contestacao_infracao/contestacao"   -H "Authorization: Bearer $TOKEN"   -F "procuracao=@/caminho/procuracao.pdf"   -F "crlv=@/caminho/crlv.pdf"   -F "cnh=@/caminho/cnh.pdf"   -F "alegacao2=@/caminho/alegacao2.pdf"   -F "ait=@/caminho/ait.pdf"   -F "ait_radar_expirado=@/caminho/ait_radar.pdf"
```

Resposta:
```json
{ "execution_id": "abc123" }
```

Depois:
```bash
curl -L -H "Authorization: Bearer $TOKEN"   -o parecer.pdf   "https://agents-store-api.alfred.vert.com.br/main-agent/v1/agent-message/download/abc123"
```
