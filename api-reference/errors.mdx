
## `docs/api-reference/errors.mdx`
```mdx
---
title: Erros comuns
---

- **400 Bad Request** — arquivo ausente/inválido.
- **401 Unauthorized** — token ausente ou inválido.
- **404 Not Found** — `execution_id` inexistente.
- **422 Unprocessable Entity** — combinação de documentos inválida para o agente.
- **500 Internal Server Error** — falha interna; tente reprocessar.
