## Pela API 
### 1. **Tarefa**
Para iniciar a tarerefa de um agent via servico se inicia com o endpoint  
`POST /main-agent/v1/agent-task/upload/contestacao_infracao/contestacao`

### 2. **Status**
Consulta o status da tarefa  
```GET /main-agent/v1/agent-message/process/{execution_id}```

### 3. **Download (PDF)**  
Após a conclusão ira ser disponibilizado um download do parecer  
```GET /main-agent/v1/agent-message/download/{execution_id}```
